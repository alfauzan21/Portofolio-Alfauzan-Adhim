Web Game Kulino

Platform web game interaktif yang dirancang untuk menghadirkan berbagai permainan yang dapat dimainkan langsung melalui browser tanpa perlu instalasi dan juga dapat melakukan pembelian di merchandise.

About This Project

Website Game Kulino adalah inovasi dalam dunia permainan berbasis browser. Misi utamanya adalah menjembatani pengalaman bermain game yang ringan dengan fitur e-commerce untuk pembelian merchandise resmi.

Dikembangkan dengan mempertimbangkan performa dan aksesibilitas, proyek ini memanfaatkan kekuatan backend Laravel dengan arsitektur frontend modern menggunakan vanilla JavaScript dan CSS tingkat lanjut.

Key Features

  • Seamless in-browser gameplay experience.
  • Integrated merchandise store and shopping cart.
  • Secure user authentication and profiles.
  • Responsive design adapted for all devices.
